# Messy Dataset Cleaner AI

## Project Description
This project automatically detects and cleans messy datasets using Python.

**Features
- Detects missing values
- Calculates missing percentage
- Drops columns with high missing values
- Fills numeric data with median
- Fills categorical data with mode
- Removes duplicate rows
- Generates cleaned dataset
- Provides cleaning report

**Technologies Used
- Python
- Pandas
- Google Colab

** Workflow
1. Load dataset
2. Analyze missing values
3. Suggest cleaning methods
4. Automatically clean data
5. Generate report
6. Save cleaned dataset
